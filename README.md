# 🚀 DevDash - Modern Admin Dashboard

Un dashboard administrativo compacto y moderno construido con un enfoque en **CSS Grid** y **diseño responsivo**. Este proyecto fue diseñado para dominar la distribución de layouts complejos y la organización visual de datos en entornos de desarrollo.

## 🛠️ Tecnologías Utilizadas

* **HTML5** - Estructura semántica.
* **CSS3 (Grid & Flexbox)** - Layout maestro y alineación de componentes.
* **Font Awesome** - Iconografía profesional.
* **Google Fonts** - Tipografía "Inter" para legibilidad máxima.

## 🎯 Objetivos de Aprendizaje Logrados

1.  **CSS Grid Mastery:** Uso de `grid-template-areas` para una estructura de "Santo Grial" (Sidebar, Header y Main) altamente mantenible.
2.  **Arquitectura Compacta:** Optimización de `gap`, `padding` y `margin` para visualización densa de datos sin saturar al usuario.
3.  **Navegación Interna:** Implementación de anclas (`ID`) con `scroll-behavior: smooth` para una experiencia de usuario (UX) fluida.
4.  **Diseño Dark Neon:** Aplicación de variables CSS (`:root`) para gestionar una paleta de colores de alto contraste.
